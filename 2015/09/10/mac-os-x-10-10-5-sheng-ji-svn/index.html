Mac OS X 10.10.5升级SVN

By Bubble

2015-09-10 更新日期:2015-09-13

文章目錄

系统自带1.7.x，升级到1.8.x

1、Homebrew安装subversion

brew install subversion

2、删除旧版SVN的相关文件

which svn
sudo rm -rf /usr/local/bin/svn*
sudo rm -rf /usr/bin/svn*

3、创建软连接

sudo ln -s /usr/local/opt/subversion/bin/svn* /usr/bin/

4、查看版本号

svn --version

svn, version 1.8.13 (r1667537)

compiled Mar 31 2015, 15:33:52 on x86_64-apple-darwin14.1.0

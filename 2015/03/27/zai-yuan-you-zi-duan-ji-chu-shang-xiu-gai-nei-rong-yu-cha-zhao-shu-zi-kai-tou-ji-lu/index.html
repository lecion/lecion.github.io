情境：

存在字段X，将字段X的内容替换为与原有内容相关的内容

Example：

intro = ‘2014届毕业生’ 替换成 intro = ‘2014级毕业生’

update table set `intro` = replace(intro, '届', '级');

如果有批量记录，并且记录并不都是以数字开头，此时需要查找数字开头的记录。可使用ascii码来判断

where ascii(`intro`) > 47 and ascii(`intro`) < 58;

连起来

update table set `intro` = replace(intro, '届', '级') where ascii(`intro`) > 47 and ascii(`intro`) < 58;
